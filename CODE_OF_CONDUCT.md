# Code of Conduct

## Our Pledge

We pledge to make participation in this project respectful and inclusive for everyone.

## Expected Behavior

- Be respectful and constructive
- Assume positive intent
- Give and accept feedback professionally

## Unacceptable Behavior

- Harassment or hate speech
- Personal attacks or intimidation
- Sharing private information without consent

## Enforcement

Maintainers may remove, edit, or reject comments, commits, code, and other contributions that violate this Code of Conduct.

## Reporting

Report incidents to:
- **Email:** conduct@yca.software
