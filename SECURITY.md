# Security Policy

## Supported Versions

Security updates are provided for the latest module release and current `main` branch.

## Reporting a Vulnerability

Please report vulnerabilities privately and avoid opening public issues.

- **Email:** security@yca.software

Please include:
- Affected package(s)
- Reproduction steps or proof of concept
- Impact assessment
- Suggested remediation (if known)

## Response Targets

- Acknowledgment: within 3 business days
- Triage: within 7 business days
- Remediation and release: based on severity and exploitability
